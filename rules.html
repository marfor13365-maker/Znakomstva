Правила сервиса Blizko

Используя приложение Blizko, вы соглашаетесь с настоящими правилами.

1. Ответственность за контент

Blizko — это площадка, предоставляющая пользователям возможность создавать профили, публиковать фото, видео, тексты и обмениваться сообщениями. Сервис не проверяет и не модерирует весь пользовательский контент заранее и не несёт ответственности за материалы, размещённые пользователями.

Каждый пользователь самостоятельно несёт полную ответственность за контент, который он публикует, отправляет или иным образом размещает на платформе, включая соответствие законодательству и правам третьих лиц.

2. Запрещённый контент и поведение

  • Материалы, нарушающие законодательство РФ или страны пребывания пользователя
  • Контент с участием несовершеннолетних в любой форме
  • Разжигание ненависти, дискриминация, угрозы, домогательства
  • Мошенничество, спам, распространение вредоносного ПО
  • Нарушение прав интеллектуальной собственности третьих лиц

3. Модерация

Администрация оставляет за собой право удалять контент и блокировать аккаунты, нарушающие данные правила, без предварительного уведомления.

4. Возраст пользователей

Сервис предназначен для лиц старше 18 лет. Регистрируясь, пользователь подтверждает, что достиг совершеннолетия.

5. Персональные данные

Обработка персональных данных осуществляется в соответствии с Федеральным законом №152-ФЗ «О персональных данных». Данные используются исключительно для работы сервиса и не передаются третьим лицам без согласия пользователя, за исключением случаев, предусмотренных законом.

6. Оплата дополнительных аккаунтов

Оплата за создание дополнительного аккаунта на устройстве производится через Telegram-бот. Оплата не подлежит возврату после выдачи кода доступа. Код одноразовый и действителен только для устройства, с которого была создана заявка.

7. Изменения правил

Администрация может изменять данные правила. Актуальная версия всегда доступна по этой ссылке.
